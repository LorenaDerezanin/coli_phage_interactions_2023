# Track P

Track P turns the locked Track G v1 feature configuration into an interactive digital phagogram and a panel coverage
heatmap for live demos.

Outputs are written under `lyzortx/generated_outputs/track_p/digital_phagogram/`:

1. `tp01_digital_phagogram.html`
2. `tp01_digital_phagogram_bundle.json`
3. `tp01_digital_phagogram_summary.json`

Track P also emits a coverage heatmap under `lyzortx/generated_outputs/track_p/panel_coverage_heatmap/`:

1. `tp02_panel_coverage_heatmap.html`
2. `tp02_panel_coverage_heatmap_bundle.json`
3. `tp02_panel_coverage_heatmap_summary.json`
